## Chat bot task
#### Download project
```
git clone https://github.com/armenstepanyan/react-chat-bot.git
cd react-chat-bot
```
Install dependencies
```
npm install
```
or
```
yarn
```

### building prod version
```
npm run build
```

### Start prod version
```
cd build
yarn serve
```

[Online Demo](https://chess.orgfree.com/task/index.html) of prod version
